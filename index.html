Noch wieviel mal schlafen?

Meine Kinder fragen ständig, wie viel mal sie noch schlafen müssen, bis sie Geburtstag haben. Mit dieser App können sie ganz einfach nachschauen, wieviele Tage es noch sind bis zu einem bestimmten Ereignis.

Features

  • Erfassen von neuen Ereignissen
  • Jedes Ereignis kann einem Familienmitglied zugeordnet werden
  • Anzeigen einer Liste mit allen Ereignissen
  • Filtern der Ereignisliste nach Person
  • Natürlich ist alles responsive

Benutzte Technologien

  • Grails Framework
  • Bulma CSS Framework
